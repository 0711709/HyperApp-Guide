
# HyperApp Guide

## HyperApp 是什么

HyperApp 是一个基于 SSH 和 Docker 的自动化部署工具， 您可以在 Store 页面点击任意应用，将其部署到您的服务器上。

### 支持的 Linux 系统

* Ubuntu 14，16 ✅
* CentOS 7 ✅
* Debian 8 ✅

注：CentOS 6 也是支持的，但是有些厂商自带的内核版本较低导致不支持 Docker

#### 测试过的厂商

* Google CloudEngine
* AWS EC2
* Linode (支持 CentOS6)
* Vultr
* DigitalOcean
